# Resume-Driven Adaptive Interview Practice System

## Overview
This project is a web-based application designed to help students and job seekers prepare for technical interviews in a personalized way. Unlike traditional interview platforms that provide generic questions, this system adapts the interview based on the candidate’s resume, skills, and performance.

Users can upload their resume in PDF format, and the system extracts key information such as technical skills and experience. Based on this, it recommends a suitable job role and conducts an adaptive interview where question difficulty changes dynamically according to user performance.

At the end of the interview, the system provides detailed feedback highlighting strengths, weaknesses, and areas for improvement.

---

## Features
- Resume upload and processing (PDF format)  
- Skill extraction using text processing techniques  
- Job role recommendation using TF-IDF and Cosine Similarity  
- Adaptive interview system with dynamic difficulty adjustment  
- Performance-based feedback generation  
- Strength and weakness analysis  

---

## Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Python (Flask)  
- Database: MySQL  
- Machine Learning / NLP: TF-IDF, Cosine Similarity  
- File Processing: PDF text extraction  

---

## Objective
To provide a personalized interview preparation platform that adapts to a user's skills and helps improve their technical interview performance through real-time feedback.
